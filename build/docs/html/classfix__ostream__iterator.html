Detailed Description

template<typename _Tp, typename _CharT = char, typename _Traits = std::char_traits<_CharT>>
class fix_ostream_iterator< _Tp, _CharT, _Traits >

Provides output iterator semantics for streams.

This class provides an iterator to write to an ostream. The type Tp is the only type written by this iterator and there must be an operator<<(Tp) defined.

Template Parameters
_TpThe type to write to the ostream.
_CharTThe ostream char_type.
_TraitsThe ostream char_traits.
